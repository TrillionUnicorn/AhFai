## 🎯 HOW TO RUN

### Development:
```bash
bun install
bun run dev
# Visit http://localhost:5173
```

### Testing:
```bash
# Install Playwright browsers
bunx playwright install --with-deps

# Run all tests
bun run test

# Run in UI mode
bun run test:ui

# Run specific test
bunx playwright test complete-verification.spec.ts
```

### Build:
```bash
bun run build
bun run preview
```

### Deploy:
```bash
# Deploy to Cloudflare Pages
wrangler pages deploy .svelte-kit/cloudflare

# Or deploy to Vercel
vercel --prod
```
